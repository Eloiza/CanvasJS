Descrição

Ao selecionar um Poligono com a ajuda do menu Polígonos, este será gerado na moldura abaixo. Ao clicar com o botão esquerdo do mouse no centro de um segmento é possível mover-lo, caso o clique for próximo as pontas do segmento é possível alterar a posição desta terminação. Ao clicar com o botão do mouse direito um segmento de reta é dividido em dois, as operções descritas anteriormente se aplicam também a estas novas retas. O código fonte desta página encontra-se disponível no Github e pode ser acessado clicando aqui!

Clique no botão para desenhar um polígono no canvas
